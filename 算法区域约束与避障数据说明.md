### 2.6 算法避障规则（城市）

```
对于每个障碍物 o = (cx, cy, cz, w, h, d):
  x_min = cx - w/2,  x_max = cx + w/2
  y_min = cy - h/2,  y_max = cy + h/2
  z_min = 0,          z_max = d

  避障条件：无人机路径点 (px, py, pz) 满足以下任一：
    px < x_min OR px > x_max  （X方向在包围盒外）
    py < y_min OR py > y_max  （Y方向在包围盒外）
    pz > z_max                （飞行高度高于建筑顶部）
```

### 3.6 算法避障规则（山区）

山区避障需**两层判断**：

```
第一层：包围盒粗筛
  与城市相同，检查是否在8个山体包围盒内

第二层：地形高程精筛（必须）
  对于路径上每个采样点 (px, py, pz):
    terrain_z = get_terrain_height(px, py)
    if pz < terrain_z:
        碰撞！路径不可行
    
    安全余量建议：pz >= terrain_z + 5m
```

### 4.3 碰撞检测函数

```python
import numpy as np

def check_collision(px, py, pz, obstacles):
    """检查点(px,py,pz)是否与任一障碍物碰撞"""
    for obs in obstacles:
        cx, cy, cz = obs["center"]
        w, h, d = obs["size"]
        x_min, x_max = cx - w/2, cx + w/2
        y_min, y_max = cy - h/2, cy + h/2
        if x_min <= px <= x_max and y_min <= py <= y_max and pz <= d:
            return True  # 碰撞
    return False  # 安全

def check_line_collision(p1, p2, obstacles, n_samples=20):
    """检查线段p1->p2是否与障碍物碰撞（采样法）"""
    for t in np.linspace(0, 1, n_samples):
        px = p1[0] + t * (p2[0] - p1[0])
        py = p1[1] + t * (p2[1] - p1[1])
        pz = p1[2] + t * (p2[2] - p1[2])
        if check_collision(px, py, pz, obstacles):
            return True
    return False
```

## 五、物资配送约束

### 5.1 无人机参数（默认）

| 参数 | 值 |
|------|-----|
| 最大载重 | 5.0 kg |
| 最大航程 | 15.0 km |
| 最大速度 | 60.0 km/h |

### 5.2 配送流程

```
服务区（起点）→ 飞行路径（避障）→ 救援点（终点）
```

- 无人机从**服务区**起飞
- 沿路径飞行，必须**避开所有障碍物**
- 到达**救援点**投送物资
- 原路返回或规划返程路径
